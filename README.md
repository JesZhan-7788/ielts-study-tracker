# IELTS 备考打卡

一个面向手机端的 IELTS 六个月备考计划互动网页，用于展示每日计划、打卡追踪和分类断档提醒。

## 功能

- 4 × 7 天数网格，按未开始、进行中、已完成显示状态
- 每日任务打卡，并记录实际完成日期
- 点击任务文字直接编辑
- 添加带“自建”标记的临时替代或加练任务
- 左右滑动、方向键或按钮切换前后日期
- 可调 3 / 5 / 7 / 10 天的分类断档提醒
- 优先使用 `window.storage(shared=false)`，普通浏览器自动回退到 `localStorage`
- 阶段数据与用户状态分离，后续阶段只需追加静态数据

当前包含阶段一完整数据：2026-07-08 至 2026-08-04，共 28 天、53 个原计划任务。

## 本地运行

需要 Node.js 22.13 或更高版本，以及 pnpm。

```bash
pnpm install
pnpm dev
```

生产构建与测试：

```bash
pnpm build
pnpm test
```

## 添加新阶段

在 `app/plan-data.ts` 的 `planData.phases` 数组中追加同结构数据即可。总览阶段选择、详情导航、完成度和提醒逻辑会自动适配，无需修改页面逻辑。

## 数据存储

计划原始数据保留在代码中；用户的勾选、完成日期、文字修改和自建任务按天保存，键格式为：

```text
{phaseId}:day:{dayIndex}
```

所有状态仅保存在当前设备浏览器，不包含账号系统或跨设备同步。
