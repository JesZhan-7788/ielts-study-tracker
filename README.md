# IELTS 备考打卡

一个面向手机端的 IELTS 六个月备考计划互动网页，用于展示每日计划和打卡追踪。

## 功能

- 4 × 7 天数网格，按未开始、进行中、已完成显示状态
- 每日任务打卡，并记录实际完成日期
- 点击任务方框或文字打卡，点击右侧铅笔按钮编辑
- 添加带“自建”标记的临时替代或加练任务
- 左右滑动、方向键或按钮切换前后日期
- 任务统一为基础、阅读、听力、写作、口语五类
- 使用浏览器 `localStorage` 在同一设备和浏览器中持久保存记录
- 阶段数据与用户状态分离，后续阶段只需追加静态数据

当前包含阶段一完整数据：Day 1–5 保留 2026-07-08 起的真实日期，Day 6–30 按实际学习节奏推进，共 30 天、59 个原计划任务。

## 本地运行

需要 Node.js 22.13 或更高版本，以及 pnpm。

```bash
pnpm install
pnpm dev
```

生产构建与测试：

```bash
pnpm build
pnpm test
```

## 添加新阶段

在 `app/plan-data.ts` 的 `planData.phases` 数组中追加同结构数据即可。总览阶段选择、详情导航和完成度会自动适配，无需修改页面逻辑。

## 数据存储

计划原始数据保留在代码中；用户的勾选、完成日期、文字修改和自建任务按天保存，键格式为：

```text
{phaseId}:day:{dayIndex}
```

所有状态仅保存在当前设备的同一浏览器和同一域名下，不包含账号系统或跨设备、跨浏览器同步。建议使用 Safari、Chrome 或添加到主屏幕，并避免无痕模式。
